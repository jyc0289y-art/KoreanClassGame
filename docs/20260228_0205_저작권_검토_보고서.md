## 2. 저작권 검토 평가

### 2-1. 위험도 평가

| 등급 | 평가 |
|------|------|
| **치환 전** | 🔴 **고위험** — 실제 등록상표 19개 직접 사용. 상표권자의 사용 중지 요청/법적 조치 가능 |
| **치환 후** | 🟢 **저위험** — 모든 상표명이 1~2자 변경된 가명으로 대체. 직접적 상표 침해 해소 |

### 2-2. 법적 안전성 분석

#### ✅ 안전하다고 판단되는 이유

1. **직접적 상표 사용 없음**: 등록상표를 그대로 사용하지 않으므로 상표법상 "상표의 사용"에 해당하지 않음
2. **혼동 가능성 최소화**: 가명은 원본과 1~2자만 다르지만, 게임 맥락에서 실제 브랜드와의 혼동이 일어날 가능성 낮음 (교육용 게임)
3. **상업적 이용 의도 없음**: 특정 브랜드의 고객 유인 목적이 아닌 한국어 학습 시나리오 구성 목적
4. **패러디/교육 목적**: 한국 문화 체험 시뮬레이션으로 교육적 목적이 명확

#### ⚠️ 잔여 위험 요소

1. **연상 가능성**: BTX→BTS 연상은 의도적이나, 이것이 "상표의 희석(dilution)"으로 해석될 수 있는 극히 낮은 가능성 존재
2. **퍼블리시티권**: 실제 아티스트를 직접 언급하지 않지만, BTX가 BTS를 가리킨다는 것이 명확한 경우 퍼블리시티권 논란 가능 (매우 낮은 위험)
3. **지역별 법규 차이**: 한국/일본/국제법상 기준이 다를 수 있음

### 2-3. 종합 의견

> **현재 치환 수준으로 일반적인 상표권 침해 리스크는 실질적으로 해소되었습니다.**
> 교육용 비상업 게임의 특성상 상표권자가 법적 조치를 취할 가능성은 극히 낮습니다.
> 다만 상업화 시에는 법률 전문가의 검토를 권장합니다.
