## 1. HOBIS 디자인 감성 분석

### 1.1 색상 체계 (CSS Variables)

| 변수명 | 색상값 | 용도 |
|--------|--------|------|
| `--hobis-green` | `#00ff33` | 주요 액센트, 데이터 텍스트, 실행 버튼 |
| `--hobis-cyan` | `#00f7ff` | 헤더, 패널 코너, 활성 탭, 보조 액센트 |
| `--hobis-bg` | `#0a0a0c` | 전체 배경 (거의 순수 블랙) |
| `--hobis-panel` | `#111418` | 패널 배경 (약간 밝은 다크) |
| `--hobis-border` | `#2a3b47` | 기본 테두리 (다크 블루그레이) |
| `--hobis-alert` | `#ff3300` | 경고/삭제 (밝은 레드) |
| `--hobis-warn` | `#ffcc00` | 주의/노랑 (옐로) |
| `--input-bg` | `#0f1215` | 입력 필드 배경 |

### 1.2 타이포그래피

| 용도 | 폰트 | 스타일 |
|------|------|--------|
| 제목/버튼/헤더 | `Orbitron, sans-serif` | 대문자, 기하학적, 미래지향 |
| 본문/데이터 | `"Share Tech Mono", monospace` | 모노스페이스, 기술적 |
| 섹션 헤더 | `Orbitron` 1rem | ALL CAPS, 시안 색상, 점선 하단 테두리 |

### 1.3 UI 패턴

| 요소 | 설계 |
|------|------|
| **패널** | 다크 배경 + 1px 테두리 + inset box-shadow + **코너 브라켓** (::before/::after 10px 시안 L자) |
| **탭** | 활성=시안 배경 0.15 투명도+시안 텍스트+글로우, 비활성=다크+뮤트 |
| **실행 버튼** | 그린 배경 0.1 투명도 + 그린 1px 테두리 + 그린 텍스트 (Orbitron) |
| **입력 필드** | 다크 배경 + 뮤트 테두리 + 그린 텍스트 + 시안 포커스 |
| **서브존** | 점선(dashed) 시안 또는 옐로 테두리로 데이터 영역 구분 |
| **로그** | 모노 폰트, 한 줄씩 누적, 타임스탬프 |

### 1.4 핵심 감성 키워드

- **군사 커맨드 콘솔** (Military HUD)
- **코너 브라켓 장식** (Sci-Fi 조종석)
- **ALL CAPS 섹션 제목** ("DECAY OPS", "MISSION LOG")
- **네온 그린/시안 발광** (다크 배경 위 고대비)
- **상태 표시기** ("QSA Online", "Ready for Calculation")
- **모노스페이스 데이터** (기술적 신뢰감)

## 2. SeouLink 적용 — 디자인 토큰

HOBIS의 군사적 감성을 유지하면서, 한국어 학습 어드벤처 맥락에 맞게 "커맨드 센터에서 서울 미션을 수행하는" 콘셉트로 재해석한다.

### 2.1 색상 팔레트

```
// ── HOBIS 계열 (게임 UI 공통) ──
--sl-green:    #00ff33    // 주요 상호작용 (HOBIS 그린)
--sl-cyan:     #00f7ff    // 헤더, 패널 코너, 활성 상태
--sl-bg:       #0a0a0c    // 전체 배경
--sl-panel:    #111418    // 패널 배경
--sl-border:   #2a3b47    // 기본 테두리
--sl-alert:    #ff3300    // 위험/잠김
--sl-warn:     #ffcc00    // 주의/경고
--sl-input:    #0f1215    // 입력 배경

// ── SeouLink 캐릭터 계열 (HOBIS 감성에 맞게 조정) ──
--sl-yuko:     #ff3366    // 유코 (핫핑크 → 네온 레드핑크)
--sl-ami:      #cc44ff    // 아미 (오키드 → 네온 퍼플)
--sl-rui:      #00ffcc    // 루이 (틸 → 네온 민트)
--sl-tester:   #ffcc00    // 테스터 (골드 → HOBIS 워닝 옐로)

// ── 맵 전용 ──
--sl-terrain:  #0d1a0d    // 맵 배경 (다크 그린-블랙)
--sl-road:     #1a2a1a    // 도로 (어두운 그린 그레이)
--sl-building: #1a1a2a    // 건물 (어두운 블루 그레이)
--sl-water:    #0a1a3a    // 수역 (다크 네이비)
```

### 2.2 타이포그래피

| 용도 | 폰트 | 비고 |
|------|------|------|
| UI 헤더/버튼 | `Orbitron` | HOBIS 동일, ALL CAPS |
| 게임 데이터/HUD | `"Share Tech Mono"` | HOBIS 동일, 모노 |
| 한국어 본문 | `"Noto Sans KR"` | 대화/미션 텍스트 |
| 일본어 본문 | `"Noto Sans JP"` | 번역/부제 |

> **원칙**: UI 프레임은 Orbitron/Share Tech Mono (HOBIS), 콘텐츠 텍스트는 Noto Sans (가독성)

## 3. 타이틀 화면 리디자인

### 3.1 현재 → HOBIS 스타일

| 요소 | 현재 | HOBIS 적용 |
|------|------|-----------|
| 배경 | `0x0a0a2e` 단색 + 별 파티클 | `#0a0a0c` + 스캔라인 오버레이 + 그리드 패턴 |
| 타이틀 | "안녕, 서울" 핑크 42px | `#00f7ff` 시안 + 텍스트 글로우 (`0 0 10px`) |
| 부제 | "アンニョン、ソウル" 오키드 | `#00ff33` 그린, Share Tech Mono |
| 태그라인 | 뮤트 블루 | `#2a3b47` 뮤트, 대시 라인 구분 |

### 3.2 캐릭터 선택 UI

```
┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐
  OPERATOR SELECT / オペレーター選択

  [■ YUKO]  [□ AMI]  [□ RUI]  [□ TESTER]
   ユコ       アミ      ルイ     テスター

  ── OPERATOR INTEL ──────────────────
  NAME: 유코 / ユコ
  ORIGIN: 후쿠오카 / 福岡出身
  CLASS: 패션잡지 기자 / ファッション誌記者
  ─────────────────────────────────────
  LV.1  EXP: ████░░░░ 0/100  ⬡ 50
└─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘
```

- **패널 스타일**: HOBIS 패널 (`--sl-panel` 배경 + 코너 브라켓)
- **캐릭터 아이콘**: 네온 색상 원형, 선택 시 시안 글로우 링
- **정보 표시**: "OPERATOR INTEL" 섹션 (HOBIS 헤더 스타일, 점선 구분)
- **레벨/경험치**: Share Tech Mono, 프로그레스 바 = 그린 블록

### 3.3 버튼

```
┌──────────────────────────────────┐
│    ▶ DEPLOY / ゲームスタート      │  ← 그린 테두리 + 그린 텍스트
└──────────────────────────────────┘
┌──────────────────────────────────┐
│    ◆ CHAPTER SELECT / 챕터選択   │  ← 시안 테두리 + 시안 텍스트
└──────────────────────────────────┘
```

- HOBIS EXECUTE 버튼과 동일 스타일
- 메인 액션 = 그린, 보조 = 시안
- Orbitron 폰트, 대문자

## 6. 캐릭터 스프라이트 리디자인

### 6.1 현재 → HOBIS 스타일

현재 캐릭터들은 **동그란 파스텔 컬러 원형**. HOBIS 감성에 맞게 **도트 아트 스타일의 전술 오퍼레이터** 느낌으로 변경.

| 캐릭터 | 현재 색상 | HOBIS 색상 | 비주얼 변경 |
|--------|----------|-----------|-----------|
| 유코 | `#ff69b4` 핫핑크 원 | `#ff3366` 네온레드 | 원형 + 시안 아이 바이저(눈 부분) + 하단 그린 라인 |
| 아미 | `#da70d6` 오키드 원 | `#cc44ff` 네온퍼플 | 동일 패턴, 퍼플 바디 |
| 루이 | `#00ced1` 틸 원 | `#00ffcc` 네온민트 | 동일 패턴, 민트 바디 |
| 테스터 | `#ffd700` 골드 원 | `#ffcc00` HOBIS 옐로 | 별 모양 아이콘 → 육각형 + "QA" 마킹 |

### 6.2 스프라이트 구성 (32×32)

```
현재:                    HOBIS 적용:
  ████                     ┌──┐
 ██○○██                   │⬡⬡│ ← 시안 바이저 (눈)
 ██▽▽██     →            ├──┤
  ████                    │▓▓│ ← 캐릭터 컬러 바디
                          └┬┬┘ ← 그린 하단 라인
```

- 둥근 원형은 유지하되, **시안 바이저 라인** (눈 부분)과 **그린 하단 디테일**로 HOBIS 요소 추가
- NPC도 동일한 바이저+하단 라인 패턴

### 6.3 NPC 스프라이트

| NPC | 현재 | HOBIS 적용 |
|-----|------|-----------|
| 이현정 | 오렌지 사각 + 베이지 얼굴 | `#ffcc00` 워닝 옐로 + 시안 바이저 |
| 김유석 | 블루 사각 | `#00f7ff` 시안 + 그린 바이저 |
| 최석 | 라임 사각 | `#00ff33` 그린 + 시안 바이저 |
| 상점NPC | 골드 사각 | `#ffcc00` + "SHOP" 마킹 |

## 7. 맵/터레인 리디자인

### 7.1 색상 체계

현재의 위성뷰 스타일을 유지하되, 색조를 HOBIS 다크+네온으로 전환. **군사 위성 정찰 뷰** 느낌.

| 지형 | 현재 | HOBIS 적용 | 비고 |
|------|------|-----------|------|
| 상업지구 | `#b8b0a0` 베이지 | `#1a2020` 다크 틸 | 건물 = 시안 아웃라인 |
| 주거지 | `#9aaa8a` 녹회 | `#0d1a0d` 다크 그린 | 건물 = 뮤트 그린 점 |
| 공원/녹지 | `#5aac4f` 밝은초록 | `#0a330a` 딥 그린 | 나무 = `#00ff33` 0.3 글로우 |
| 도로 대로 | `#888888` 회색 | `#1a2a2a` 다크 + 시안 중앙선 | 차선 = `#00f7ff` 0.2 점선 |
| 도로 소로 | `#666666` | `#151f1f` | 테두리 = `#2a3b47` |
| 수역 (한강) | `#1a3a6a` 네이비 | `#0a1540` 딥 네이비 | 파문 = 시안 0.05 |
| 해안 (하카타만) | 블루 그라데이션 | `#050a20` → `#0a1a3a` | 스캔라인 오버레이 |
| 건물 (채움) | 다양한 밝은 색 | `#111418` + 네온 아웃라인 | 옥상 = 뮤트 색상, 테두리 = 시안/그린 |
| 인도 | `#bbbbbb` | `#1a2a2a` + 점 패턴 | |
| 공항 터미널 | 베이지/회색 | `#111418` + 시안 존 라인 | 군사기지 감성 |
| 활주로 | `#3a3a3a` | `#0a0a0c` + 그린 중심선 + 시안 엣지라인 | |

### 7.2 건물 시각

```
현재:                         HOBIS 적용:
┌──────┐                     ┌─ ─ ─ ─┐
│██████│ ← 단색 채움          │ ◇    ◇│ ← 시안 코너 브라켓
│██████│                     │       │ ← 다크 배경 + 네온 아웃라인
│██ ██│ ← 창문               │ ■  ■ │ ← 시안 창문 (글로우)
│█ █ █│                     │  ▤   │ ← 그린 문
└──────┘                     └─ ─ ─ ─┘
```

- 건물은 **다크 바디 + 네온 아웃라인** (HOBIS 패널과 동일 원리)
- 각 건물 타입마다 다른 네온 색상:
  - 지하철역: 시안 `#00f7ff`
  - 음식점: 레드 `#ff3300`
  - 카페: 워닝 옐로 `#ffcc00`
  - 올리브숲: 그린 `#00ff33`
  - 학원: 퍼플 `#cc44ff`
  - 공항: 시안 `#00f7ff`

### 7.3 가로수/식생

- 현재: 밝은 녹색 원형
- HOBIS: `#00ff33` 0.2 투명도 원형 + 가운데 밝은 점 (글로우 효과)
- 공원 나무군: 오버랩하는 원형들, `#00ff33` 0.15 글로우

### 7.4 거리 표지판

```
현재: 파란 배경 흰 글씨
HOBIS: 다크 배경 + 시안 테두리 + 그린 텍스트 (Share Tech Mono)

┌────────────────┐
│ 홍대입구역 🚇   │ ← 시안 테두리, 그린 텍스트
│ HONGDAE STN    │ ← Share Tech Mono
```

## 12. 디자인 원칙 요약

1. **다크 우선**: 배경은 항상 `#0a0a0c` ~ `#111418`. 밝은 배경 없음
2. **네온 액센트**: 정보는 `#00ff33` (그린), 구조는 `#00f7ff` (시안), 위험은 `#ff3300`
3. **코너 브라켓**: 모든 패널/다이얼로그에 HOBIS 코너 브라켓 장식
4. **모노스페이스 데이터**: 수치/상태 정보는 반드시 Share Tech Mono
5. **ALL CAPS 헤더**: 섹션 제목은 Orbitron + 대문자 + 시안
6. **군사 용어**: "DEPLOY", "OPS", "INTEL", "COMM LINK", "TRANSIT COMMAND"
7. **글로우 효과**: 중요 요소에 `0 0 10px` 텍스트/박스 글로우
8. **점선 구분**: 섹션 구분은 실선이 아닌 dashed 라인

> **핵심 콘셉트**: "서울로 파견된 오퍼레이터가 커맨드 콘솔에서 미션을 수행하며 한국어를 배운다"
